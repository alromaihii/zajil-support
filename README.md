Real-time GCC news intelligence app for iPhone and iPad.

## About

Zajil-News is a real-time news monitoring app built for the Gulf Cooperation Council (GCC) region. It uses AI to score every event by severity (1-10), tracks live financial markets, delivers push notifications for critical developments, and provides AI-generated situation briefs.

## Features

- AI-scored news events with interactive map
- Live market data: indices, stocks, commodities, forex
- AI-generated situation briefs and threat assessments
- Push notifications for critical events
- Home screen widgets
- Live TV news channels
- Bilingual support (English / Arabic)
