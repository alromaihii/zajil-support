## About

Zajil-News is a real-time situational awareness monitor for the Gulf Cooperation Council region — Bahrain, Saudi Arabia, UAE, Kuwait, Qatar, and Oman. It aggregates headlines from Reuters, Bloomberg, AP, Al Arabiya, and other trusted sources, then uses AI to score each event for severity and generate concise summaries.

Built for analysts, traders, journalists, and anyone who needs to stay informed about the region.

## Features

- **AI-scored events** with severity ratings (1–10) and geographic mapping
- **Live market data**: indices, stocks, commodities, forex — filter All / GCC / International
- **AI situation briefs**: daily AI-generated summaries of overnight developments
- **Push notifications** for critical events (no account required)
- **Home screen widgets**: Top Alert, Threat Level, Alerts List, and full Dashboard
- **Live TV news channels** embedded in-app
- **Bilingual**: full English and Arabic support with proper RTL layout

## Device Requirements
